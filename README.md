# Lab Kesehatan Daerah — Dashboard (Prototype Statis)

Dashboard ini dibuat sebagai **website statis** (tanpa backend) dengan **data mock** yang relevan untuk kebutuhan kantor Lab Kesehatan Daerah.

## Cara Menjalankan
1. Buka file:
   - `index.html`
2. Bisa langsung melalui browser:
   - double click `index.html`
   - atau drag-drop ke browser

## Fitur Utama
- Ringkasan KPI (Sampel Masuk, Pemeriksaan Selesai, SLA Tepat Waktu, Rata-rata Waktu Proses)
- Rekap Tabel Pemeriksaan (dengan pencarian & sorting)
- Grafik Tren (harian/mingguan)
- Status Antrian & Backlog
- Ekspor CSV sesuai filter

## Struktur Project
- `index.html`
- `style.css`
- `app.js`
- `data/mock-data.js`

## Catatan
Data menggunakan contoh mock untuk memudahkan pengujian tampilan & logika. Nantinya dapat dihubungkan ke Excel/Google Sheets atau database.
